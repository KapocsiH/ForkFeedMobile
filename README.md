# ForkFeedMobile

A ForkFeed Mobile A ForkFeed webapplikáció androidos portja, amely egy receptmegosztó közösségi platform.

Mivel ugyanazt a backend API-t használja mint a webalkalmazás, ezért a funkciók zöme megegyezik.

Amiben többet tud:
* Bevásárlólista
* Kamera használata képfeltöltéshez

---

## Telepítés

Két módon lehet telepíteni:
APK telepítéssel vagy Visual Studio-ból futtatva.

---

### 1. Telepítés APK fájlból (Release)

Lépések:

* Töltse le a .apk fájlt a releases tabról
* Engedélyezze eszközén az ismeretlen forrásból származó appok futtatását
* Telepítse az appot

---

### 2. Futtatás Visual Studio-ból (Release módban)

#### Követelmények:

* Visual Studio (MAUI támogatással)
* Android SDK telepítve
* Emulator vagy Android telefon

#### Lépések:

* Nyissa meg a leklónozott projektet Visual Studio-ban
* Állítsa át a "Build Configuration"-t Release-re
* Válasszon eszközt: Emulator (pl. Pixel) vagy saját telefon (USB debugging bekapcsolva)

4. Indítás:

* Kattintson a Run gombra

Az app build után elindul az eszközön.
